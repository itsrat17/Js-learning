# JavaScript Learning Journey 🚀

A comprehensive collection of JavaScript learning materials, examples, and practice exercises organized for structured learning.

## 📚 Project Structure

```
.
├── 01_basics/           # Fundamental JavaScript concepts
├── 02_basics/           # Data structures and objects
├── 03_basics/           # Functions and advanced concepts
├── 04_control_flow/     # Control flow and conditional logic
├── 05_iterations/       # Loops and iteration methods
└── README.md           # This file
```

## 🎯 Learning Path

### 1. Fundamentals (`01_basics/`)
- Variables and declarations
- Data types and type conversion
- String manipulation
- Numbers and mathematical operations
- Date handling in JavaScript
- Comparison operations

### 2. Data Structures (`02_basics/`)
- Arrays and array methods
- Objects and object manipulation
- Object-oriented concepts

### 3. Advanced Concepts (`03_basics/`)
- Functions (regular and arrow functions)
- Scope and closures
- IIFE (Immediately Invoked Function Expressions)
- Interview preparation notes

### 4. Control Flow (`04_control_flow/`)
- Conditional statements
- Switch cases
- Truthy and falsy values

### 5. Iterations (`05_iterations/`)
- Various loop types
- Array iteration methods
- Advanced iteration patterns

## 🛠️ Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/itsrat17/Js-learning.git
   cd Js-learning
   ```

2. Open any JavaScript file in your preferred editor or IDE

3. Run individual files using Node.js:
   ```bash
   node 01_basics/01_variables.js
   ```
